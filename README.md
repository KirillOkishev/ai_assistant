# Проект: "ИИ-ассистент для риэлторской компании"

## Описание проекта

Проект направлен на создание интеллектуального помощника (ИИ-ассистента) для риэлторской компании, который будет интегрирован в Telegram. Ассистент способен вести осмысленный диалог с клиентами, отвечать на вопросы по базе знаний, а также записывать клиентов на встречи и автоматически добавлять их в Google Календарь. Это решение позволит оптимизировать взаимодействие с клиентами, сократить время на рутинные задачи и повысить эффективность работы риэлторов.

Ссылка на проект - https://web.telegram.org/k/#@OtbasyHomeBot

## Функционал

### 1. **Telegram-бот**
   - Бот полностью интегрирован в Telegram.
   - Позволяет клиентам взаимодействовать с компанией через удобный мессенджер.
   - Обеспечивает быстрые и точные ответы на запросы пользователей.

### 2. **Осмысленное ведение диалога**
   - Используется нейросеть для анализа входящих сообщений и формирования контекстуальных ответов.
   - Бот способен поддерживать естественный диалог, учитывая предыдущие сообщения пользователя.

### 3. **Ответы на вопросы по базе знаний**
   - Создана структурированная база знаний, содержащая информацию о:
     - Услугах компании.
     - Ценообразовании.
     - Доступных объектах недвижимости.
     - Условиях сотрудничества.
   - Бот автоматически извлекает информацию из базы знаний и предоставляет клиентам точные ответы.

### 4. **Запись на встречу**
   - Клиент может запросить запись на встречу через бота.
   - Бот предлагает доступные временные слоты и фиксирует выбор клиента.
   - Информация о встрече автоматически добавляется в Google Календарь риэлтора.

---

## Используемые сервисы и технологии

### 1. **Qwen**
   - Применяется для создания системного промпта, который задает поведение нейросети.
   - Используется для формирования и управления базой знаний.
   - Обеспечивает высокую точность и релевантность ответов.

### 2. **Савви (Suvvy.ai)**
   - Сервис используется для создания и настройки ИИ-ассистента.
   - Предоставляет инструменты для обучения модели на основе данных компании.
   - Обеспечивает интеграцию с Telegram и другими внешними сервисами.

### 3. **Telegram**
   - Платформа для развертывания бота.
   - Обеспечивает удобный интерфейс для взаимодействия с клиентами.

### 4. **Google Календарь**
   - Используется для автоматической записи встреч.
   - Интеграция позволяет синхронизировать расписание риэлторов и избежать конфликтов времени.

## Преимущества проекта

1. **Автоматизация рутинных задач**:
   - Бот берет на себя обработку стандартных запросов, что освобождает время риэлторов для более важных задач.

2. **Повышение качества обслуживания**:
   - Клиенты получают мгновенные и точные ответы на свои вопросы.

3. **Удобство для клиентов**:
   - Возможность записаться на встречу в любое время через Telegram.

4. **Экономия ресурсов**:
   - Снижение затрат на поддержку клиентов за счет автоматизации.

5. **Масштабируемость**:
   - Решение легко адаптируется под растущие потребности компании.

---

## Этапы реализации

1. **Анализ требований**:
   - Изучение бизнес-процессов компании.
   - Определение ключевых функций бота.

2. **Создание базы знаний**:
   - Сбор и структурирование информации о компании и услугах.
   - Настройка базы знаний в Qwen.

3. **Разработка бота**:
   - Создание прототипа бота с использованием Suvvy.ai.
   - Настройка диалогового интерфейса.

4. **Интеграция с Telegram и Google Календарь**:
   - Подключение API Telegram для обработки сообщений.
   - Настройка синхронизации с Google Календарем.

5. **Тестирование**:
   - Проверка работоспособности всех функций.
   - Корректировка модели и базы знаний на основе обратной связи.

6. **Запуск и поддержка**:
   - Внедрение бота в рабочие процессы компании.
   - Мониторинг работы и внесение улучшений.
